# 账户系统

CoS:

+ 用户
    * 用户注册, 邮箱必填
    * 用户名/邮箱登录
    * 微信登录
    * 忘记密码, 通过邮箱重置密码
    * 重置密码
    * 用户信息
        - 昵称
        - 邮箱
    * 绑定
+ 图书驿站
    * 申请入驻
    * 登录
    * 驿站员工管理
+ 平台
    * 管理员账号
    * 登录

## Requirements

- [JWT](https://jwt.io/)
- [Shiro](https://shiro.apache.org/)
